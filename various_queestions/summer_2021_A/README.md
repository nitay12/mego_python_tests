# שאלה 5
נתונה הפעולה what:

```py
def what(arr):
    m = arr[0]
    s = 0
    for i in range(len(arr)):
        s = s + arr[i]
        if s > m:
            m = s
        if s < 0:
            s = 0
    return m
```
